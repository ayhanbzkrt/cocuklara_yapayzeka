Pixel ve Byte'ın Macerası

Pixel ve Byte, Bilginova gezegeninin Bilgi Ormanı'nda bir maceraya atılıyorlar. Ormanda dolaşırken, Pixel farklı renklerdeki parlak meyveleri görüyor: bazıları kırmızı (integers), bazıları mavi (strings) ve diğerleri yeşil (floats). Byte, her meyvenin farklı bir "veri tipi" olduğunu açıklıyor.

Etkinlikler

Bu etkinlikte temel Python veri tiplerini öğreneceksiniz.

sayi = 5
print("Tam Sayı:", sayi)
isim = "Pixel"
print("Metin:", isim)
ondalik = 3.14
print("Ondalıklı Sayı:", ondalik)

Değişkenleri tanımlayın ve print fonksiyonunu kullanarak ekrana yazdırın.

agac = "Yeşil Ağaç"
print("Değişken:", agac)
hayvan = "Tavşan"
bitki = "Meşe Ağacı"
print("Hayvan:", hayvan)
print("Bitki:", bitki)

Toplama, çarpma ve mantıksal operatörleri öğrenin.

toplam = 5 + 3
print("Toplama İşlemi Sonucu:", toplam)
carpma = 4 * 2
print("Çarpma İşlemi Sonucu:", carpma)
bolme = 8 / 2
print("Bölme İşlemi Sonucu:", bolme)
print("5 büyük müdür 3'ten?", 5 > 3)

Koşullu ifadeler kullanarak kararlar alın.

yol = "sol"
if yol == "sol":
    print("Sol yolu seçtiniz!")
elif yol == "sağ":
    print("Sağ yolu seçtiniz!")
else:
    print("Bilinmeyen bir yol seçtiniz.")

While ve for döngüleri kullanarak tekrarlanan görevleri yapın.

adim = 0
while adim < 5:
    print("Labirentte ilerliyoruz, adım:", adim)
    adim += 1

for i in range(5):
    print("Labirentte adım:", i)

Fonksiyonlar tanımlayın ve kullanın.

def gizem_coz(gizem):
    print(gizem, "çözüldü!")

gizem_coz("Labirentin Gizemi")

def gorev_yap(gorev, kisi):
    print(gorev, kisi, "yapıyor!")

gorev_yap("Labirentin Gizemi", "Pixel")
gorev_yap("Bilgi Ormanı", "Byte")

Listeler oluşturun ve üzerlerinde işlemler yapın.

bilgiler = ["Yıldızlar", "Galaksiler", "Uzay Gemileri"]
print("İlk bilgi:", bilgiler[0])
print("Tüm bilgiler:", bilgiler)

for bilgi in bilgiler:
    print("Bilgi:", bilgi)

Sınıflar tanımlayın ve nesne yönelimli programlamayı öğrenin.

class Robot:
    def __init__(self, isim):
        self.isim = isim
    def selamla(self):
        print("Merhaba, ben", self.isim)

pixel_robot = Robot("Pixel")
pixel_robot.selamla()

class TemizlikRobotu(Robot):
    def temizle(self):
        print(self.isim, "temizlik yapıyor.")

temizlik_robotu = TemizlikRobotu("Byte")
temizlik_robotu.selamla()
temizlik_robotu.temizle()

Fonksiyonlara argümanlar geçin ve return ifadesini kullanın.

def toplama(sayi1, sayi2):
    return sayi1 + sayi2

sonuc = toplama(3, 5)
print("Toplama sonucu:", sonuc)

Listeler üzerinde ileri seviye işlemler yapın.

renkler = ["mavi", "kırmızı", "yeşil", "sarı"]
renkler.sort()
print("Sıralı renkler:", renkler)

en_sevdigim = renkler[0]
print("En sevdiğim renk:", en_sevdigim)

bilgiler = ["Yıldızlar", "Galaksiler", "Uzay Gemileri"]
bilgiler.append("Kara Delikler")
print("Yeni Bilgiler:", bilgiler)

bilgiler.remove("Yıldızlar")
print("Güncellenmiş Bilgiler:", bilgiler)
